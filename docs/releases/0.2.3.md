# whytab 0.2.3

## 中文

0.2.3 是桌面布局与右键交互修复版本，不改变数据结构、同步协议或账号数据。

### 修复内容

- 桌面导航固定在屏幕最左或最右边缘，不再受页面最大宽度影响而向中间偏移。
- 工作区使用对称安全边距。切换左侧、右侧、始终显示、自动隐藏或隐藏时，页面中心线保持不变。
- 在较窄桌面窗口中保留导航与内容之间的安全间距，避免悬浮导航遮挡小组件。
- 主页使用统一的右键捕获入口，确保小组件、布局编辑模式和主页空白区域都能打开菜单。
- 主页文件夹恢复右键编辑入口。
- 移除会导致 Chrome 中右键菜单瞬间关闭的窗口失焦监听。

### 数据安全

- 数据结构版本继续保持为 `1`。
- 不修改同步、登录、账号隔离或云端数据逻辑。
- 不执行用户数据迁移、删除、覆盖或重建。

### 使用

- 在线版：打开当前官方在线地址并刷新页面。
- Chrome / Edge：下载 `whytab-0.2.3-chrome-web-store.zip`，解压并加载该目录。
- 已加载本地目录：更新目录后，在扩展管理页点击一次“重新加载”。

## English

Version 0.2.3 fixes desktop workspace geometry and context-menu reliability. It does not change the data schema, sync protocol, or account data.

### Fixes

- Pins desktop navigation to the far left or right viewport edge instead of the centered content container.
- Uses symmetric safety gutters so changing navigation side or visibility never shifts the workspace center.
- Preserves clearance between floating navigation and content on narrower desktop windows.
- Uses one capture-phase context-menu entry point for widgets, layout editing, and the empty Home workspace.
- Restores right-click editing for Home folders.
- Removes window-blur dismissal that could immediately close a context menu in Chrome.

### Data Safety

- The data schema remains at version `1`.
- Sync, authentication, account isolation, and cloud-data behavior are unchanged.
- No user-data migration, deletion, overwrite, or rebuild is performed.

### Install

- Web app: open the current official hosted URL and refresh.
- Chrome / Edge: download and extract `whytab-0.2.3-chrome-web-store.zip`, then load that directory.
- Existing unpacked installation: replace the local files and click **Reload** once on the extensions page.
