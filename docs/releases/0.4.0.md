# whytab 0.4.0

## 中文

0.4.0 是一次以内容、材质和可靠交互为核心的界面升级。它重做快捷图标、小组件、资源中心、右键菜单与手机滚动体验，但不修改数据结构、登录流程、账号隔离、同步协议或本地存储逻辑。

### 图标与工作区

- 网站真实图标直接占满图标区域，不再套用额外彩色大框；主页、网站页和资源中心共用相同尺寸与圆润比例。
- 自动图标优先获取更清晰的站点资源，并保留品牌图标、内置图标和文字回退，加载失败时不会留下空白。
- 页面改为壁纸优先的中性通透材质，时间、搜索、导航、卡片和弹层保持一致层级，不再给不同小组件随机染色。
- 桌面导航固定在屏幕边缘并垂直居中，主内容不因切换左侧、右侧或隐藏方式而横向跳动。

### 小组件与交互

- 重新整理 12 个小组件的标题、字号、留白、数据层级和空状态，使天气、日历、待办、倒计时、专注、照片、汇率和灵感等组件拥有清晰而不同的内容结构。
- 网站、文件夹、页面空白区和小组件右键菜单统一改为顶层 Portal，避免被卡片、模糊层或页面层级遮挡。
- 小组件尺寸选择改为三行真实比例预览；倒计时新增正式编辑弹窗，不再调用浏览器原生输入框。
- 保留鼠标、触屏和键盘组件排序。布局编辑模式提供独立拖动手柄、即时预览和移动完成反馈。
- 卡片透明度滑杆现在实时控制组件材质；图标尺寸继续同时作用于主页和网站页。

### 壁纸与移动端

- 新增 20 张 whytab 原创壁纸，分为日系、原创动漫、猫咪和酷感四组，并保留精选与自定义分类。
- 每张新壁纸提供 `1600x900` 桌面文件和 `720x1280` 手机文件；资源中心使用懒加载，手机无需下载横屏大图。
- 壁纸筛选、上传、壁纸集和每日轮换统一收进可滚动资源中心，并修复内容较多时筛选按钮被压缩而无法点击的问题。
- 手机内容改为在底部导航上方独立滚动，导航不再覆盖卡片或文字；页面切换会自动回到顶部，最后一个组件可完整滚入安全区。

### 数据与兼容性

- 数据结构版本继续保持为 `1`，不会因为更新重建、清空或覆盖已有用户数据。
- 不修改注册、登录、退出、账号数据隔离、自动同步、合并同步、本地覆盖云端或云端覆盖本地逻辑。
- 继续保留更新前备份、JSON 导入导出和本地优先使用方式。

## English

Version 0.4.0 is a content, material, and interaction-focused interface update. It redesigns shortcut icons, widgets, the resource center, context menus, and mobile scrolling without changing the data schema, authentication flow, account isolation, sync protocol, or local storage behavior.

### Icons and Workspace

- Real website artwork now fills the icon itself instead of sitting inside an extra colored container. Home, Websites, and the resource center share the same size and geometry.
- Automatic icon loading prioritizes sharper site assets while retaining brand, built-in, and text fallbacks so a failed request never leaves an empty tile.
- The workspace uses neutral wallpaper-aware translucent materials with a consistent hierarchy across time, search, navigation, cards, and dialogs.
- Desktop navigation remains pinned to the viewport edge and vertically centered without shifting the main workspace when its side or visibility changes.

### Widgets and Interaction

- Refines typography, spacing, hierarchy, and empty states across all 12 widgets while giving weather, calendar, todos, countdowns, focus, photos, rates, and quotes distinct content layouts.
- Website, folder, page, and widget context menus now render through a top-level portal so cards, blur layers, and stacking contexts cannot hide them.
- Widget sizing uses three clear, true-ratio previews. Countdown creation now uses an in-product editor instead of a native browser prompt.
- Mouse, touch, and keyboard sorting remain supported through dedicated handles, a live drag preview, and completion feedback.
- The card-transparency slider now controls the material live, while the icon-size slider continues to apply identically on Home and Websites.

### Wallpapers and Mobile

- Adds 20 original whytab wallpapers across Japanese, original illustrated, cat, and cinematic collections, alongside Featured and custom categories.
- Every new wallpaper includes a `1600x900` desktop file and a `720x1280` phone file. The resource center lazy-loads previews, and phones avoid downloading landscape originals.
- Style filtering, uploads, wallpaper collections, and daily rotation share one scrollable resource center, including a fix for compressed, unclickable filters in long dialogs.
- Mobile content now scrolls in a viewport above the bottom navigation, preventing the bar from covering cards or text. Page changes reset to the top and the final widget remains fully reachable.

### Data and Compatibility

- The data schema remains at version `1`; this update does not rebuild, clear, or overwrite existing user data.
- Registration, sign-in, sign-out, account isolation, automatic sync, merge sync, local-to-cloud overwrite, and cloud-to-local overwrite behavior are unchanged.
- Pre-update backups, JSON import/export, and local-first use remain available.
