# whytab 0.5.0

## 中文

0.5.0 是一次完整的小组件体验升级。12 个小组件不再重复同一种卡片内容结构，而是根据天气、时间、任务、记录和数据换算等不同用途分别设计。数据结构、注册登录、账号隔离、同步协议和本地存储逻辑均未改变。

### 小组件设计

- 天气：加入实时状态、风速、降水和六日趋势，天气信息可以从当前概况自然扫读到后续变化。
- 日历：展开尺寸组合日期撕页和月历，紧凑尺寸保留当天最重要的信息。
- To Do：使用环形完成进度、任务计数和独立输入区，进度与下一步操作不再挤在同一行。
- 倒计时：以轨道数字突出距离目标日的天数，并将日期和状态分层显示。
- 专注：使用计时环、当前周期和阶段标记，开始、暂停与复位保持直接可操作。
- 照片：有照片时使用完整画面和底部说明；空状态使用轻量画框提示上传。
- 每日灵感：改为编辑排版风格，以引号、正文和署名形成清晰阅读层级。
- 世界时钟：组合模拟钟面、主时区数字时间和城市时区列表。
- 便签：使用横线纸、定位点、日期和字数，让记录状态更明确。
- 年度进度：以年度百分比、当前天数和 52 周点阵同时表达整体与阶段进度。
- 计算器：增加结果屏、双输入区和分段运算符，适合鼠标与触屏操作。
- 中行汇率：采用对齐的数据表头、币种标记和换算结果区，提高数字扫读效率。

### 尺寸与交互

- 紧凑、标准和展开尺寸会根据卡片实际宽度折叠次要信息，保留核心操作。
- 小组件右键设置、拖拽排序、尺寸切换、刷新和编辑行为保持可用。
- 支持键盘语义、减少动态效果和减少透明度等现有辅助功能设置。

### 数据安全

- 数据结构版本仍为 1，不需要新增迁移步骤。
- 本地 IndexedDB、账号数据隔离和云同步行为保持不变。
- 更新不会清除、覆盖或重新归属已有快捷方式、小组件、笔记、待办、倒计时、设置或布局。

## English

Version 0.5.0 is a complete widget experience refresh. All 12 widgets now use content-specific structures for weather, time, tasks, notes, and conversion data instead of repeating one generic card template. The data schema, registration and sign-in flow, account isolation, sync protocol, and local-storage behavior are unchanged.

### Widget design

- Weather adds live status, wind, precipitation, and a six-day trend for progressive scanning.
- Calendar combines a tear-off date panel with the month grid while compact sizes retain the essential day information.
- To Do separates completion progress, task counts, input, and the working list into a clearer task workspace.
- Countdown uses an orbital day marker with layered status and date information.
- Focus uses a timer ring, session state, and stage markers while keeping start, pause, and reset directly operable.
- Photo presents a full image with a caption when configured and a lightweight framed upload state when empty.
- Daily Inspiration adopts an editorial quote layout with clear quotation, copy, and attribution hierarchy.
- World Clock combines an analog dial, primary digital time, and a compact list of city time zones.
- Memo uses ruled-paper cues, a pin marker, date, and character count to clarify recording state.
- Year Progress combines the annual percentage, current day, and a 52-week dot matrix.
- Calculator adds a result display, paired inputs, and segmented operators suitable for pointer and touch input.
- BOC Rates uses aligned headers, currency marks, and a dedicated conversion result area for faster numeric scanning.

### Sizing and interaction

- Small, medium, and wide cards adapt to their actual container width and progressively remove secondary details.
- Widget context settings, drag ordering, size controls, refresh actions, and editing remain available.
- Existing keyboard semantics, reduced-motion support, and reduced-transparency support remain in place.

### Data safety

- The data schema remains at version 1 and requires no new migration.
- Local IndexedDB storage, account isolation, and cloud-sync behavior are unchanged.
- Updating does not clear, overwrite, or reassign existing shortcuts, widgets, notes, todos, countdowns, settings, or layouts.
