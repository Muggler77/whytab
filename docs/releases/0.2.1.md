# whytab 0.2.1

## 中文

0.2.1 重做了 whytab 的主页、导航和小组件界面，同时保持现有本地数据与云端同步格式兼容。

### 新界面

- 首页改为居中的工作台布局，移除重复的“搜索和常用入口”标题，减少无效留白。
- 统一快捷图标容器、图片加载状态和文字区域，修复图片、占位符与标题重叠的问题。
- 12 个小组件使用一致的标题、图标、操作按钮、间距和信息层级。
- 桌面端使用稳定双列布局；平板和手机端自动切换为适合触控的单列布局。
- 深色、浅色、不同图标尺寸和不同小组件尺寸均保持可读。

### 布局编辑

- 点击首页工具栏的“编辑布局”按钮后，可以拖动快捷图标和小组件重新排序。
- 触屏设备支持先选择项目、再选择目标位置，避免滚动时误触拖动。
- 拖动状态、目标位置和完成状态都有明确视觉反馈。

### 自定义页面

- 左侧导航可以创建工作、学习、生活等个人页面。
- 每个个人页面使用独立快捷方式分组，可以直接添加网站和文件夹。
- “网站”和“工具”页面可以隐藏或恢复；“主页”始终保留。
- 删除个人页面只删除导航入口，不删除对应分组和网站数据。

### 交互修复

- 修复主页和小组件区域右键菜单失效或重复触发的问题。
- 右键菜单会自动限制在当前窗口内，靠近屏幕边缘时不会被裁切。
- 优化网站图标回退速度；加载成功后只显示真实图标，失败时显示稳定占位符。

### 数据安全与兼容

- 数据结构版本继续保持为 `1`，0.2.1 不执行破坏性数据升级。
- 旧版本的快捷方式、文件夹、待办、笔记、倒计时、图标大小、小组件顺序和尺寸会保留。
- 新增的个人页面和页面显示设置会随现有设置一起保存和同步。
- 未登录用户的数据仍保存在当前浏览器的 IndexedDB；登录后的同步逻辑没有更改。

## English

Version 0.2.1 redesigns the whytab home workspace, navigation, shortcuts, and all widgets while keeping the existing local data and cloud sync format compatible.

### Highlights

- A centered, quieter home workspace with consistent shortcut sizing and no icon/label overlap.
- Twelve redesigned widgets with unified headers, controls, spacing, and responsive behavior.
- An explicit layout editing mode for reordering shortcuts and widgets, including a touch-safe two-step interaction.
- User-created navigation pages backed by shortcut groups.
- Optional hiding of the Websites and Tools pages; Home remains protected.
- Custom page deletion removes only the navigation entry and preserves its group and website data.
- Restored and boundary-clamped right-click menus.
- Faster favicon fallback behavior and stable loading states.

### Compatibility

- The data schema remains at version `1`.
- Existing shortcuts, folders, todos, notes, countdowns, icon size, widget order, and widget sizes are preserved.
- Sync behavior and account authentication are unchanged.

## Installation

- Web and mobile: open [why-tool.com](https://why-tool.com/).
- Chrome or Edge: download `whytab-0.2.1-chrome-web-store.zip`, extract it, enable Developer mode, choose **Load unpacked**, and select the extracted folder.
- Existing unpacked installations: replace the installation folder with the 0.2.1 files and click **Reload** on the browser extensions page.
