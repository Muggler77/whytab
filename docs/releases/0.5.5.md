# whytab 0.5.5

## 中文

0.5.5 完善网页安装、移动端图标和公开隐私说明，不修改数据结构或同步协议。

- 增加 192px、512px 和 maskable PWA 图标，满足 Chromium PWA 安装图标要求。
- Apple 主屏幕图标升级为 192px。
- 网页启动后请求浏览器持久化本地存储，降低本机数据被系统空间回收清理的概率。
- 登录注册面板增加公开隐私与数据说明入口。
- 移除扩展清单中未使用的 `storage` 和 `alarms` 权限，只保留当前功能实际需要的权限。
- 数据结构版本仍为 1，最低同步兼容版本仍为 0.5.2。

## English

Version 0.5.5 improves web installation, mobile app icons, and public privacy disclosure without changing the data schema or sync protocol.

- Adds 192px, 512px, and maskable PWA icons for Chromium installability.
- Upgrades the Apple home-screen icon to 192px.
- Requests persistent browser storage on web startup to reduce the chance of local data eviction.
- Adds a public privacy and data-handling link to the login and registration panel.
- Removes the unused `storage` and `alarms` extension permissions and keeps only permissions required by current features.
- The data schema remains version 1 and the minimum sync-compatible version remains 0.5.2.
