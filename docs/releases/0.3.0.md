# whytab 0.3.0

## 中文

0.3.0 是 whytab 的完整界面与交互重设计版本。它统一了桌面、平板和手机体验，但不改变数据结构、账号隔离、同步协议或云端数据。

### 全新设计系统

- 以单一 `ui-v030` 样式系统替代旧版三层 CSS 覆盖，统一颜色、文字、间距、圆角、阴影和交互状态。
- 采用中性高对比画布与多色功能强调，避免整页单一色调和小组件互相混用视觉语言。
- 时间与搜索严格位于页面中心轴，桌面内容保持对称边距，导航切换左右时不移动主内容。
- 桌面导航固定在视口边缘并垂直居中；手机和平板使用带安全区的底部导航。

### 页面和组件

- 主页使用自适应 12 列组件网格，宽屏自动紧密排布，手机端自动切换单列。
- 天气、日历、待办、倒计时、专注、照片、汇率、每日灵感、时钟、便签、年度进度和计算器全部统一重做。
- 主页与网站页使用相同的快捷图标尺寸和圆润比例；文件夹、Dock、空状态和拖拽状态同步更新。
- 工具箱、资源中心、壁纸库、图标库、页面管理、快捷导航编辑、设置和时区选择采用统一布局。
- 账号与同步页面重新排版，增加密码显示与隐藏控制，不显示服务地址或访问密钥。

### 交互与可访问性

- 保留并验证主页组件拖拽、网站页右键菜单、组件右键尺寸预览和布局撤销流程。
- 手机端可见操作区至少为 `44px`，底部导航预留安全区且页面无横向滚动。
- 增加跳到主要内容、可见键盘焦点、弹窗语义、Escape 关闭和减少动态效果支持。
- 深色与浅色主题同时覆盖普通页面和 Portal 弹窗。

### 性能与数据安全

- 生产 CSS 从旧版约 `249 KB` 降至约 `66 KB`，减少约 74%。
- 数据结构版本继续保持为 `1`。
- 不修改登录、账号隔离、合并同步、覆盖同步或本地存储逻辑。
- 不执行用户数据删除、重置、覆盖或重建；现有主题、壁纸、图标尺寸和布局设置继续保留。

## English

Version 0.3.0 is a complete UI and interaction redesign for whytab. It unifies desktop, tablet, and mobile experiences without changing the data schema, account isolation, sync protocol, or cloud data.

### New Design System

- Replaces three overlapping legacy stylesheets with one `ui-v030` system for color, typography, spacing, radius, elevation, and interaction states.
- Uses a neutral high-contrast canvas with distinct functional accents instead of one dominant color family.
- Places time and search on the exact viewport center line while preserving symmetric desktop content gutters.
- Keeps desktop navigation pinned to the viewport edge and vertically centered; mobile and touch tablets use a safe-area-aware bottom bar.

### Pages and Components

- Introduces a responsive 12-column widget grid with dense wide-screen placement and a single-column mobile layout.
- Redesigns weather, calendar, todos, countdowns, focus, photo, rates, quote, world clock, memo, year progress, and calculator widgets.
- Uses the same shortcut icon geometry across Home and Websites, including folders, Dock, empty states, and drag states.
- Unifies Tools, Resource Center, wallpaper and icon libraries, page management, shortcut editing, Settings, and timezone selection.
- Reworks Account and Sync and adds a password visibility control without exposing service URLs or access keys.

### Interaction and Accessibility

- Verifies widget drag-and-drop, website context menus, widget size previews, and layout undo behavior.
- Keeps visible mobile targets at least `44px`, respects safe areas, and prevents horizontal overflow.
- Adds a skip link, visible keyboard focus, dialog semantics, Escape dismissal, and reduced-motion support.
- Applies both dark and light themes to the app and portal-rendered dialogs.

### Performance and Data Safety

- Reduces production CSS from about `249 KB` to about `66 KB`, a reduction of roughly 74%.
- Keeps the data schema at version `1`.
- Does not change authentication, account isolation, merge sync, overwrite sync, or local-storage behavior.
- Performs no user-data deletion, reset, overwrite, or rebuild; existing theme, wallpaper, icon-size, and layout choices remain intact.
